# Nas Kurdish Learning

بەرنامەیەکی فێربوونی کوردی بۆ منداڵان. بەکارهێنانی یاری فلاشکارد و یاری هاوتاکردن بۆ فێربوونی پیتەکانی زمانی کوردی.

## تایبەتمەندییەکان

*   لیستی یارییەکان (فلاشکارد، هاوتاکردن، دەنگی پیتەکان، نووسین، خوێندن، بینگۆ)
*   یاری فلاشکارد: پیشاندانی پیتەکانی کوردی
*   یاری هاوتاکردن: دۆزینەوەی هەمان پیت
*   ڕووکارێکی سادە و ڕەنگاوڕەنگ گونجاو بۆ منداڵان

## تەکنەلۆژیاکان

*   Flutter
*   Dart

## چۆنیەتی کارپێکردن

1. دڵنیابە لەوەی Flutter دابەزێنراوە.
2. پڕۆژەکە کلۆن بکە یان دابەزێنە.
3. بڕانە ناو فۆڵدەری پڕۆژەکە: `cd nas_kurdish_learning`
4. پەڕگەکان بەدەست بهێنە: `flutter pub get`
5. بەرنامەکە کارپێبکە: `flutter run`
